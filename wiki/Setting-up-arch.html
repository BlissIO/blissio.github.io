This guide assumes you have already installed archlinux with a minimal install without Xorg set up

I prefer not to use the archinstall script that comes with the iso. This isn't because I am some elitist and the such
it is just that I prefer knowing what I have on my distro to the maximum that's why I use minimal distros like arch, Gentoo, and kiss Linux...

I think that the command pacstrap itself is an arch installer script because when you go to distros like Gentoo and kiss you
realize just how much stuff is automated with pacstrap.

To install a minimal install of Xorg on achrlinux run the following command:

pacman -S --noconfirm xorg-server xorg-xinit xorg-xkill xorg-xsetroot xorg-xbacklight xorg-xprop

Now after this I prefer installing fonts so you could use your web browser and other programs I always go for the following packages:

pacman -S noto-fonts noto-fonts-emoji noto-fonts-cjk ttf-jetbrains-mono ttf-joypixels ttf-font-awesome

These packages always took care of any problems I had with Xorg or any other program not starting (of course just the font problem).

Next here are some basic packages that any archinux user would need (neofetch not included :p)

pacman -S ffmpeg imagemagick fzf man-db unclutter xclip git zip unzip unrar p7zip xdotool \
dosfstools ntfs-3g libnotify dunst jq networkmanager wpa_supplicant rsync pamixer libconfig

After that I install an aur helper because of course why would you use arch without using the aur? (This is just a joke btw relax)

cd Â£HOME
mkdir .config
git clone https://aur.archlinux.org/pikaur.git
cd pikaur
makepkg -fsri
cd $HOME

This code puts pikaur's sourcecode inside your .config folder in your home directory.

After installing an aur helper I would install libxft-bgra-git because I use dwm and need such packages for emojis to work.

pikaur -S libxft-bgra-git

Now ! let's get into the real stuff to make Xorg work you need to set up your .xinit folder.

Here I will give the example of installing the awesome window manager since most people going to such lengths
of installing arch would use a window manager. And that the awesome window manager is easy to use and understand

touch .xinitrc/span>

Now open .xinitrc in your favorite text editor and place the following text:

exec awesome

This enables you to run the awesome window manager after you run the "startx" command

Now if you want to know what to put in the .xinitrc file to start your chosen program please head onto the archlinux wiki and type your chosen desktop environment or window manager in the search bar most of the time you are gonna have the needed text in their page.
If not a quick google search should fix the issue.

Please have fun with your arch install tinker and explore to your heart's content !! Do not be afraid of breaking it it is part of the learning process.
